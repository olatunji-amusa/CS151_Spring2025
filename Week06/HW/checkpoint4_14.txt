Write an if/else statement that assigns 0.10 to commissionRate unless sales
is greater than or equal to 50000.00, in which case it assigns 0.20 to
commissionRate.

ANSWER BELOW
*******************************************************************************
if (sales >= 50000.00)
{
    commissionRate = 0.20;
}
else
{
    commissionRate = 0.10;
}
